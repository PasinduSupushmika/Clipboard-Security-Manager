INTRODUCTION:
-------------
Clipboard Security Manager (CSM) is an enterprise-grade desktop utility that protects your sensitive clipboard data from malicious applications, accidental leaks, and unauthorized viewing. 

Rather than sending raw text or images directly into your vulnerable memory buffers (like RAM), CSM immediately encrypts your data using military-grade AES-256-GCM cryptography. It then only decrypts and pastes your text when YOU authorize it!

ADVANTAGES OF USING CSM:
------------------------
1. Total Clipboard Protection: All copied data is locked locally; background applications/malware cannot read your clipboard.
2. Military-Grade Encryption: Employs NIST-approved AES-256-GCM encryption natively.
3. Brute-Force Immunity: Backed by Argon2 natively, defending your password through memory-hard verifications.
4. Intelligent Leak-Proofing: Unauthenticated users hitting "Paste" will only ever receive a clean warning message rather than bleeding your raw password hashes into your documents.
5. Email Alert Watchdog: Sends instant Email Alerts tracking IP/Date logging if any incorrect attempts are initiated on your machine.
6. Easy Interface: Modern, low-footprint Sci-Fi User Interface using fluid CustomTkinter dialogs.

HOW TO INSTALL & RUN IT:
------------------------
Because this application is bundled natively into an executable wrapper, installing it is incredibly straightforward even if you have no programming background!

Step 1: Locate the `ClipboardSecurityManager.exe` inside the `dist` folder.
Step 2: Double-click it. (If Windows Defender asks, click "More Info" and "Run Anyway").
Step 3: A neon Cyber Setup Wizard will appear only on the very first time you open it. 
Step 4: Type in a Master Password you can remember and finalize the Initialization sequence.
Step 5: You're safe! 

HOW TO USE IT:
--------------
1. Simply copy text as you always do natively (Selecting it, then hitting CTRL+C or Right-Click -> Copy).
2. When you want to retrieve the secure data, navigate to your target document and hit CTRL+V.
3. The CSM native pop-up block will interrupt the workflow. Provide your Master Password natively.
4. CSM will automatically decrypt your data and securely paste it seamlessly!

NOTE: To manage passwords or view logs, just double-click the green padlock icon currently located silently in your Windows Bottom-Right Taskbar.
